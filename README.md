# 这是一个肆仟-智能协同云图库后端代码

# 采用技术栈
* springboot
* mysql
* mybatisPlus
* redis+caffeine多级缓存
* Jsoup数据抓取
* cos对象存储
* ShardingSphere分库分表
* Sa-token权限控制
* WebSocket双向通信
* Disruptor高性能无锁队列
* 阿里百炼AI绘图大模型
