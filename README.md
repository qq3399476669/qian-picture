# 这是一个肆仟-智能协同云图库后端代码

# 采用技术栈
* springboot
* mysql
* mybatisPlus
* redis+caffeine多级缓存
* Jsoup数据抓取
* cos对象存储
* ShardingSphere分库分表
* Sa-token权限控制
* WebSocket双向通信
* Disruptor高性能无锁队列
* 阿里百炼AI绘图大模型

# 项目功能
* 首页图片展示
* 上传图片、url上传图片
* 批量抓取图片
* 我的空间
* 团队空间
* 管理员可进行用户管理、图片管理、空间管理
* 可根据图片类别、标签进行检索
* 可根据图片主色调进行检索
* 对空间进行分析
* 对图片进行AI扩图
* 以图搜图
* 团队空间内对图片进行多人协同编辑
* 等等功能
